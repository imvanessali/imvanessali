JSON Format Guide

Upload a .json file to create your own flashcard deck. Your file should follow this structure:

{
  "_title": "word",
  "_blur": ["meaning", "sentence"],
  "cards": [
    {
      "word": "abandon",
      "phonetic": "/əˈbændən/",
      "meaning": "放弃",
      "sentence": "Never abandon your dreams."
    }
  ]
}

Three config fields

_title

Which field to show as the card heading (always visible). Defaults to the first field if omitted.

_blur

Array of field names to blur-hide initially. Swipe down to reveal. All other fields stay visible.

cards

Array of card objects. Each card can have any fields you want — they'll be rendered automatically.

Only this object-based schema is supported.
